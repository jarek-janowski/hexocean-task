# HexOcean task

## TOOLS

- create-react-app
- typescript
- React Hooks
- sass/scss
- redux-form & redux-form-input-masks

## HOW TO USE

1. Clone repository
2. cd hexocean-task
3. Run `npm install`
4. Run `npm start`

## Available scripts

`npm install` - install the dependencies in the local node_modules folder 

`npm start` - runs development mode

`npm run build` - runs build process for production
